# amso-frontend-box

Данный пройект реализует базовый фреймворк для реализации frontend проектов.

## Цель

Реализовать универсальное решение по разработки React проектов с поддержкой BFF и SSR.

## Использование 
Использование в режиме разработчика
```
npm ci
npm run dev 
```
Использование в производственном режиме 
```
npm ci
npm run build
npm run start 
```
### Архитектура frontend
При разработке frontend части приложения используйте FSD архитектуру и развивайте все модули как изолированные части приложения.

### Архитектура backend 
При разработке backend части используется полная поддрежка type-script
